# Hello, new site

*Posted 2026-05-21*

I scrapped the old Jekyll site and rebuilt this one from scratch. It is, as you can probably tell, a retro Windows 95 desktop running entirely in your browser.

## Why

The old site was a stock Jekyll theme I'd shoved my name into in 2019. It worked, but it wasn't *mine*. I wanted something that felt like a place I'd actually want to hang out — opinionated, a little weird, and built by hand.

## What's here

- **README.txt** — the short version of who I am.
- **My Computer** — projects I'm proud of. Star counts are pulled live from the GitHub API, so they stay fresh without me ever rebuilding the site.
- **Blog** — what you're reading now.
- **Terminal** — type `help` to see the dozen commands. Try `sudo make me a sandwich`.

## How it's built

Zero frameworks. No build step. Vanilla HTML, CSS, and JS, hand-rolled. Hosted on GitHub Pages. The whole site is small enough that you could read every line of source in an afternoon, and that's the point.

If you want to see how it works, the source is on [GitHub](https://github.com/djcopley/djcopley.github.io). Open the inspector. Poke around. That's encouraged.

## What's next

More posts (probably about whatever I'm building at the time), more apps on the desktop, and maybe a guestbook if I can figure out how to do it without a backend. If you have ideas, the email's on the **README.txt** window.
